Programs of my principal github repository

Project 1 : Auto email from excel file

This is a Python program that uses the XLRD module (so you will have to install this module in order to use the program) to read emails, names ..., from an Excel file and uses this information to send personalized emails automatically.

Notice: the e-mail address used by the program must have SMTP enabled.

Project 2 : Auto email attach

Same for this program except that it can send attachments, I designed this program to send hundreds of personalized resumes with one click, later I knew that my sister was sending emails for hours, then we had the idea to modify this program to personalize and send thousands of emails in a few seconds.

This program yet simple and easy to realize (it took me only one afternoon) was able to save at least 13000$ according to one company

Project 3 : Deban

This prog designed with AutoIt and Batch is about a fly (debana in arabic :p) that comes out during an adjustable interval in Batch part of the program, so goes the duration.

The second Batch file isaba deban launches several flies in screen for an amusing effect :p.

Project 4 : SDL2 installer

For those of the domain (C programmers) it is painful each time to install a new library (kind of extension of a programming language), of course I do not speak about Linux (only one line on Bash and that's it), but Windows, eh yes of the interminable copy and paste.

with this program it's history, make the changes on the Batch file indicate the path of your C/C++ compiler, the Batch script will copy the SDL2 library elements in the appropriate location, there is even an program to test if the lib was well installed.

Project 8 : Arab text to unicode converter

Some libraries do not take into account the formatting of Arabic characters (writes all the letters in their isolated form), and the online converters that I found on the internet do not, so that's why I wrote this program, just write the desired Arabic words on "arabic_test.txt" then launch the program and finally retrieve your unicode on "utf_code.txt".

Project 10 : Keyboard Racing Arabic (prototype)

Project 11 : Hijri Gregorian date converter (prototype)

The C++ version of the program on console to convert a Gregorian date into a Hijri date and vice-versa, it uses the Julian calendar (Julian as in Julius Caesar, a kind of scientific calendar used in astronomy) as an intermediary between the two mentioned calendars. Is based on the algorithm of the doctor Aly Abbara which uses the Julian calendar (Julian as Julius Caesar, a kind of scientific calendar used in astronomy) as an intermediary between the two calendars mentioned one above.

Project 12 : Date Converter (for mobile & PC)

Project 13 : Andek Lbelgha (prototype)

Project 14 : zaitoun utilities (for mobile & PC)
